# BATA 비서 (BATA Secretary)

## 제품소개서

---

## 1. 제품 개요

### 제품명
**BATA 비서** (BATA Secretary)

### 슬로건
"회의와 상담의 기록을 지능형 비서처럼 자동 정리"

### 핵심 가치
- **시간 절감**: 회의/상담 후 기록 작성 시간 80% 감소
- **정확성**: AI 기반 자동 요약으로 빠진 내용 방지
- **법적 근거**: 녹취 원본과 함께 서명된 최종 기록 자동 생성
- **보안 우선**: 민감 정보는 내부 시스템에만 저장, 외부 유출 차단

---

## 2. 주요 기능

### A. 회의 모드 (Meeting Mode)
회의 중 휴대폰 앱으로 음성 실시간 전송
- **자동 기록**: 회의 중 발언 내용 자동 인식 및 기록
- **화자 식별**: 누가 언제 무엇을 말했는지 자동 구분
- **중간 요약**: 회의 중 주요 의제, 결정사항, 할 일을 카드로 표시
- **현장 확인**: 진행자가 요약 내용을 회의 중 확인하고 수정
- **자동 문서화**: 회의 종료 후 표준 회의록 자동 생성
- **서명 완료**: 담당자 전자서명으로 법적 효력 부여

### B. 상담 모드 (Counseling Mode)
조용한 환경에서 개인 상담 내용 기록
- **저시인성**: 상담자가 휴대폰을 쳐다봐야 할 필요 없음
- **백그라운드 수집**: 상담 내용을 조용히 자동 기록
- **질의응답 구조화**: "호소 > 청취 > 조치 > 후속" 형태로 자동 정리
- **담당자 검수**: 상담 후 상담자만 기록 확인 및 수정
- **제한적 공개**: 기록은 등록된 사람(상담자/관리자)만 열람 가능
- **서명 고정**: 최종 서명 후 기록 변경 불가

## 3. 기대 효과

### 운영 효율성
| 항목 | 기존 방식 | BATA 비서 | 개선율 |
|---|---|---|---|
| 회의 후 기록 작성 시간 | 30~60분 | 5~10분 (검수만) | **80% 감소** |
| 회의록 오류 발견 | 사후 적응 | 실시간 확인 | **즉각 대응** |
| 담당자별 작업 배치 누락 | 12% | 0.5% | **95% 개선** |

### 의료/보험/교육 기관 효과
- **의료**: 환자 상담 기록 체계화, 소송 대비 완벽한 기록 보관
- **보험**: 고객 상담 내용 표준화, 분쟁 해결 시 증거 자료 완비
- **교육**: 학생 면담 기록 자동화, 진로 지도 자료 축적

## 4. 보안 및 규정 준수

### 데이터 보호 원칙
- **내부 저장**: 모든 음성, 사진, 텍스트는 서버에만 저장
- **외부 유출 금지**: 원본 데이터를 외부 AI 서비스로 전송하지 않음
- **암호화 저장**: 모든 데이터는 암호화되어 저장
- **접근 제어**: 로그인 후 권한별로만 열람 가능

### 감시 추적
- **사용 기록**: 누가 언제 어떤 기록을 봤는지 기록
- **수정 이력**: 수정된 모든 항목과 수정자 기록 자동 저장
- **삭제 정책**: 90일 후 원본 음성 자동 삭제 (기록은 보존)
- **감사 가능**: 관리자와 감사자가 모든 로그 조회 가능

### 법적 근거
- **녹취 동의**: 시작 전 동의 화면으로 법적 근거 마련
- **전자서명**: 대표/담당자의 서명으로 공식 문서화
- **타임스탬프**: 모든 기록에 생성/수정/서명 시각 기록

## 6. 시스템 구성 (기술 안내)

### 운영 환경
- **서버**: Mac mini (16GB 메모리, 512GB 저장소)
- **저장소**: 내부 네트워크 전용 또는 VPN 접속
- **사용자**: 휴대폰 앱 (iOS/Android)

### 핵심 기술
- **음성인식**: 로컬 AI 기반 (외부 API 미사용)
- **화자분리**: 누가 언제 말했는지 자동 구분
- **이미지 분석**: 화이트보드 사진에서 텍스트 자동 추출
- **자동 요약**: AI가 중요 항목만 추출

### 규모 지원
- **MVP**: 1일 1회 세션, 최대 1시간
- **확장 가능**: 일일 3~5회 세션으로 확대 가능

## 7. 도입 효과 사례 (예상)

### 병원 진료실
```
이전: 진료 후 밤새 진료기록 작성 (2시간)
이후: 귀가 후 15분 검수 및 서명만 진행
효과: 야근 80% 감소, 오류율 95% 감소
```

### 보험 상담센터
```
이전: 상담 후 기록 작성 (30분), 분쟁 발생 시 기록 재구성 (3일)
이후: 기록 자동 완성 (5분), 분쟁 시 원본 음성 + 기록으로 즉시 해결
효과: 상담 대기 시간 30% 단축, 분쟁 해결 시간 90% 단축
```

### 학교 상담실
```
이전: 면담 후 노트 정리, 3주 후 기록 작성
이후: 면담 즉시 기록 완성, 시스템에 축적
효과: 학생 지도 자료 완비, 면담 일관성 개선
```

## 11. 자주 묻는 질문 (FAQ)

### Q1. 음성 데이터가 외부로 나가나?
**A.** 절대 아닙니다. 모든 음성은 내부 서버에만 저장되며, 외부 AI 서비스로 전송되지 않습니다.

### Q2. 녹취 동의는 어떻게 받나?
**A.** 앱 시작 시 동의 화면을 표시하여 법적 근거를 마련합니다.

### Q3. 기록을 수정할 수 있나?
**A.** 최종 서명 전까지는 수정 가능하며, 서명 후에는 수정 이력만 추가 기록됩니다.

### Q4. 비용은 얼마나 드나?
**A.** 구성과 규모에 따라 다르므로 협의 필요합니다. (별도 문의)

### Q5. iOS와 Android 모두 지원하나?
**A.** MVP는 한 플랫폼부터 시작하며, 이후 양쪽 지원 예정입니다.
