# Phase 2 - 심화 분석 (Deep-Dive)

**목적:** 상세한 요구사항을 구체화하고, MVP 스펙을 명확히 하기 위한 심화 인터뷰  
**작성일:** 2026-05-08  
**상태:** 1차 답변 완료, MVP 1차 범위로 축소 정리 중
미국 주식의 종가를 확인하고 뉴스를 확인하고 이전 데이터를 분석하는데 시간이 너무 많이 걸리고 시간을 놓쳐서 자주 까먹고 분석하지 못하고 넘길때가 많아.. 
이러한 대응으로 매일 미국 주식장이 마감되어지면 그날의 뉴스를 종합해서 꼭지 뉴스 5가지를 알려주고
나스닥, S&P500, 다우지수, 주요 ETF(기술,금융,반도체,중소형주,원자재 등)의 변동성 데이터와 공탐지수, 빅스지수등도 알려주고
주요지수 혹은 주요 ETF 를 하는 사람들이 참고하고 싶은 짧은 내일의 투자전략 조언을 각 종목에 넣어주었으면해
가독성을 좋게 하기위해 RSI, 맥디, 등의 지수 변화도 보여주고 52주 최고가 최저가 대비 종가의 위치도 알려주길 바래

## 출력물 재정의 (2026-05-08)

현재 단계에서 출력물은 단순 텍스트가 아니라 "앱 위젯형 가독성"을 기준으로 한다.

### 1) 앱 화면 출력 기준
- 상단: 날짜 선택 + 10줄 내 시장 요약
- 중단: 주요 지수/ETF 카드
- 각 카드 필수 요소:
   - 종가
   - 전일 대비 변화율(색상 강조)
   - 주간 미니 그래프(7일 sparkline)
- 하단: 주요 뉴스 5개

### 2) 이메일 출력 기준
- 개발 완료 전: 전송 기능 비활성화, 우측 패널 화면만 유지
- 개발 완료 후: HTML 본문 기반 카드형 레이아웃으로 전송
- 본문 구성: 요약 카드 + 지수/ETF 카드(주간 그래프 포함) + 뉴스 카드

### 3) PDF 출력 기준
- 텍스트 나열형이 아니라 카드형 리포트
- 앱 주요 카드 구조와 동일한 정보 밀도 유지
- 지수/ETF별 주간 미니 그래프 포함

### 4) 검수 플로우
- 1차: 컴퓨터에서 앱 화면 및 HTML 미리보기 확인
- 2차: 레이아웃/가독성 확정
- 3차: 최종 승인 후 이메일 전송 활성화

## MVP 1차 범위 제안

현재 요구사항은 충분히 매력적이지만 첫 버전으로는 범위가 큽니다. 그래서 1차에서는 아래까지만 우선 구현합니다.

### MVP 1차에 포함
- 미국장 마감 후 하루 1회 데이터 업데이트
- 메인 화면 상단 날짜 + 10줄 이내 시장 요약
- 주요 지수 3개: NASDAQ, S&P 500, Dow Jones
- 핵심 ETF 소수: TQQQ, SOXL, 금융 ETF 1개, 반도체 ETF 1개 정도로 시작
- 주요 뉴스 5개 표시
- 각 카드에 종가, 전일 대비 변화율, 색상 표시
- 최근 7일 로컬 저장
- 오프라인 시 마지막 캐시 데이터 표시
- 영어 원문 기반, 앱 UI는 한국어/영어 병행 가능하도록 구조 준비

### MVP 1차에서 제외
- 3배 ETF 10개 전체
- 빅테크 7개 전체 동시 제공
- 코인, 원유, 채권, 환율, 달러지수 전체 확장
- RSI, MACD, 52주 위치
- 종목별 AI 투자전략 조언
- 즐겨찾기, 푸시 알림

### MVP 1차 이후 2차 확장
- 자산군 확대: 빅테크, 채권, 코인, 원유, 환율, 달러지수
- 기술지표 추가: RSI, MACD, 52주 최고/최저 대비 위치
- 개인화 기능: 즐겨찾기, 푸시 알림
- AI 기능: 뉴스 종합 요약, ETF/지수별 내일 전략 코멘트

- 주식 심볼: MVP 1차에서는 주요 지수 3개와 핵심 ETF 일부만 우선 제공. 확장 시 3배 ETF 10개, 빅테크 7개, 채권, 코인, 원유 추가
- 뉴스 소스: 주요 이슈 종목 또는 지수 관련 심볼 검색 기반 뉴스 수집
- 뉴스 언어: 영어 뉴스만 사용
- 추가 지표: 1차는 주요 지수 중심, 이후 VIX, 원달러 환율, 달러지수 등 확장
- 업데이트 주기: 미국 주식장 마감 후 하루 1회 업데이트

### UI/UX
- 메인 레이아웃: 날짜 선택 후 상단 10줄 요약, 그 아래 카드, 뉴스, 지표 순서
- 카드 디자인: 상세 카드 형태
- 테마: 다크 모드
- 시각화: 텍스트 + 색상 방식

### 기술
- 로컬 저장: 최근 7일 데이터 저장
- 백엔드: 처음부터 구성하되 초기에는 개인 노트북에서 실행 후 추후 서버 PC로 이전
- 오프라인: 캐시된 데이터 표시 + 오프라인 표시
- 다국어: 영어 뉴스 기반, 앱 UI는 한국어/영어 지원 방향

### 배포 & 운영
- 배포 플랫폼: 우선 테스트 버전 중심 운영
- 버전 관리: Semantic Versioning 사용
- 피드백 수집: 초기 단계에서는 별도 기능 없이 진행

### 추가 기능
- 즐겨찾기: MVP 2차로 이관
- 푸시 알림: MVP 2차로 이관
- AI 요약: MVP 2차로 이관

### 성공 기준
- MVP 기준: 주요 지수 3개, 핵심 ETF 일부, 뉴스 5개, 10줄 시장 요약, 7일 캐시, 오프라인 표시까지 동작하면 완료
- 성능 목표: 로딩 5초 이내, 메모리 50MB 이하, 배터리 최적화는 후순위

### 기타 의견
