## 1. 프로젝트 목표

### 1.1 비전
> 📱 **미국 주식 시장의 **일일 변화**를 자동으로 요약하고, 사용자가 한눈에 핵심 정보를 파악할 수 있는 모바일 앱**

### 1.2 문제 정의
- 현재 주식 정보는 복잡한 차트와 수치로 가득함
- 일반인이 하루하루의 시장 변화를 간단히 이해하기 어려움
- **해결책:** AI가 요약한 뉴스 + 주요 지표로 직관적 전달

### 1.3 핵심 가치
- **편의성:** 앱만 열면 어제 시장 상황 바로 파악
- **접근성:** 차트 없이 텍스트와 숫자로 간단히 정리
- **신뢰성:** 공개 API를 통한 실시간 정확한 데이터

## 2. 기술 스택

### 2.1 프론트엔드
| 항목 | 선택 | 이유 |
|------|------|------|
| 플랫폼 | Flutter (iOS + Android) | 크로스 플랫폼, 빠른 개발, 성능 우수 |
| 언어 | Dart | Flutter 기본 언어 |
| 상태 관리 | Provider | 간단하고 가볍고 배우기 쉬움 |
| UI 라이브러리 | Material 3 + Cupertino | 플랫폼별 네이티브 디자인 |
| 날짜/시간 | intl 패키지 | 다국어 지원, 포매팅 |

### 2.2 백엔드 (Phase 1: 직접 API 호출)
```
[Flutter App] 
    ↓
[외부 API 직접 호출] ← Phase 1에서는 이것 사용
    ├─ Finnhub API (주식 데이터)
    └─ NewsAPI (뉴스 데이터)
```

### 2.3 데이터 소스
| 데이터 | API | 요금 | 레이트 제한 |
|--------|-----|------|-----------|
| 미국 주식 | Finnhub | 무료 tier 있음 | 60회/분 |
| 금융 뉴스 | NewsAPI | 100회/일 (무료) | 필요시 유료 |
| 시장 지표 | Finnhub 포함 | 무료 | 위와 동일 |

## 3. 핵심 기능 (MVP 기능 세트)

### 3.1 Feature 1: 일일 주식 요약 대시보드
```
[화면]
┌──────────────────────────┐
│  📅 어제 (2026-05-07)     │
├──────────────────────────┤
│  Apple (AAPL)            │
│  종가: $150.25           │
│  변화: ↑ +$2.50 (+1.7%)  │
│                          │
│  S&P 500                 │
│  지수: 5,320.15          │
│  변화: ↑ +45.30 (+0.86%) │
└──────────────────────────┘
```
- **기본 정보:** 종가, 변화율, 화살표 시각화
- **대상 심볼:** AAPL, MSFT, GOOGL (MVP: 고정 3개)
- **배치 시간:** 매일 오후 5시 (한국 시간)

### 3.2 Feature 2: 주요 뉴스 5개
```
[화면]
┌────────────────────────────┐
│  📰 오늘의 주요 뉴스        │
├────────────────────────────┤
│  1️⃣ Apple announces...     │
│     (2026-05-07 13:45)    │
│                            │
│  2️⃣ Market rally as...     │
│     (2026-05-07 12:30)    │
│                            │
│  ... (3개 더)              │
└────────────────────────────┘
```
- **소스:** NewsAPI (키워드: "stock market", "US economy")
- **개수:** 5개
- **정렬:** 최신순 (publishedAt 기준)

### 3.3 Feature 3: 시장 변화 지표
```
[화면]
┌───────────────────────────┐
│  📊 시장 지표              │
├───────────────────────────┤
│  지표          변화       │
│  ─────────────────────   │
│  S&P 500    +0.86%       │
│  NASDAQ-100 +1.25%       │
│  Dow Jones  +0.42%       │
└───────────────────────────┘
```
- **포함 지표:** S&P500, NASDAQ-100, Dow Jones
- **업데이트:** 매일 시장 종가 후

### 3.4 Feature 4: 날짜 선택 (선택사항 - v1.1)
```
[UI]
┌─────────────────┐
│  < 2026-05-07 > │  ← 스와이프 또는 달력
└─────────────────┘
```
- **기본값:** 어제
- **범위:** 최대 30일 전까지

## 4. 데이터 흐름 (아키텍처)

```
┌─────────────────┐
│  사용자 앱 실행   │
└────────┬────────┘
         │
         ↓

│  1. 날짜 결정              │
│     (기본: 어제)           │

         │
         ↓

│  2. 병렬 API 호출           │
│  ├─ Finnhub (주식 데이터)  │
│  ├─ Finnhub (시장 지표)   │
│  └─ NewsAPI (뉴스)        │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│  3. 캐싱 (로컬 저장소)     │
│     SharedPreferences      │

         │
         ↓
┌────────────────────────────┐
│  4. UI 렌더링              │
│     (주식 카드 + 뉴스)     │
└─────────────────────────────┘
```

    ↓

## 5. 성공 기준 (Definition of Done)

### 5.1 기능 테스트
- ✅ 앱 시작 후 **2초 내**에 어제 주식 정보 로드
- ✅ 뉴스 **정확히 5개** 표시

- ✅ **iOS 14+** 정상 작동

- ✅ 온라인/오프라인 모두 안정적

### 5.3 UI/UX
- ✅ 한눈에 정보 파악 가능 (3초 내)
- ✅ 스마트폰 화면 크기 (4.5~6.7인치) 모두 대응

### 5.4 성능
- ✅ API 응답시간: 평균 < 2초
- ✅ 메모리 사용: < 100MB
